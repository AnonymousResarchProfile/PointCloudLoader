# 3D Scene Visualizer

Standalone scene-bundle viewer and bounding box annotation editor for point clouds.

# Installation

Use Python 3.12.4. The project supports Python `>=3.11,<3.13`, but 3.12.4 is what we worked with.

```bash
python -m pip install -r requirements.txt
```
